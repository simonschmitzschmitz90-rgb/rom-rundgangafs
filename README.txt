Römische Schnitzeljagd – OFFLINE-VERSION

Diese Version verwendet keine externen Bibliotheken mehr.
Dadurch ist sie besser für Schul-iPads und restriktive Netzwerke geeignet.

So aktualisierst du die Website:
1. ZIP entpacken
2. Alle Dateien im bestehenden GitHub-Repository ersetzen
3. Änderungen hochladen / committen
4. Die Website neu laden

Wenn weiterhin ein leeres Feld erscheint:
- Browser-Cache leeren oder Seite hart neu laden
- auf dem iPad Safari neu starten
